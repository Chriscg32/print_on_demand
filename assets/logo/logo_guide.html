Logo Variations

Your brand requires the following logo variations:

  • Primary Logo - Full color on transparent background (PNG)
  • White Logo - White version for dark backgrounds (PNG)
  • Square Logo - Optimized for social media profiles (PNG)
  • Favicon - Simplified version for browser tabs (ICO)
  • Vector Version - Scalable version for print (SVG or AI)

How to Generate Logo Variations

Prerequisites

Install ImageMagick:

  • Windows: Download for Windows
  • Mac: brew install imagemagick
  • Linux: sudo apt-get install imagemagick

Steps

  1. Copy your original logo file to this directory:
    C:\Users\chris\cgapp\print_on_demand\assets\logo\
  2. Rename it to primary_logo.png
  3. Run the appropriate script:
    • Windows: Double-click generate_logo_variants.bat
    • Mac/Linux: Run ./generate_logo_variants.sh
  4. Review the generated files and make any necessary adjustments

Vector Version Note

For the vector version (SVG), you'll need to use a vector editing program like Inkscape (free) or Adobe Illustrator. A true vector conversion cannot be done automatically from a raster image.

Logo Usage Guidelines

Clear Space

Maintain clear space around the logo equal to the height of the letter "B" in "ButterflyBlue"

Minimum Size

  • Print: No smaller than 1 inch wide
  • Digital: No smaller than 150 pixels wide

Incorrect Usage

The following uses are not permitted:

  • Stretching or distorting the logo
  • Changing the logo colors outside the approved palette
  • Placing the logo on busy backgrounds that reduce visibility
  • Adding effects like shadows or glows
